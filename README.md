# CdrTableModel4Eclipse
CDR Table Model for Eclipse TPS.

This table model is in a zip file, which is actually a folder that you can directly click open. The .geo file includes most of the actual CDR table model details.

To import to the Eclipse TPS, just go to Contouring workspace (no need to open any patient), click Structure menu (top of screen), select Creat and Manage Support Structure Models, select Manage Models.

In the newly opened window, click Import button, select the zip file you downloaded, Open. You should see a message says "Import Report: Model CDR Table Model -> **Success**".

The only thing is that the zip file name "CDR Table Model" cannot be the same as any name of your existing Support Structure Models in your list.
